# User Story: Suivi projets et affaires

## Story

**As a** chef de projet
**I want** consulter l'état de mes projets et leurs tâches
**So that** je puisse suivre l'avancement et identifier les points de blocage

## Acceptance Criteria

### AC1: Liste des projets actifs
- **Given** la question "Quels sont les projets en cours ?"
- **When** le skill génère le SQL
- **Then** le résultat liste les projets avec end_date IS NULL et leurs clients associés

### AC2: Tâches d'un projet
- **Given** la question "Quelles sont les tâches du projet X ?"
- **When** le skill génère le SQL
- **Then** le résultat joint projects_tasks_lines et projects_tasks_operations_lines

### AC3: Projets par client
- **Given** la question "Quels projets pour le client Y ?"
- **When** le skill génère le SQL
- **Then** le résultat résout projects_idcor_account vers accounts
