# User Story: Achats par fournisseur

## Story

**As a** responsable achats
**I want** analyser mes volumes et montants d'achats par fournisseur
**So that** je puisse négocier et optimiser ma base fournisseurs

## Acceptance Criteria

### AC1: Top fournisseurs par montant d'achat
- **Given** la question "Quels sont mes principaux fournisseurs cette année ?"
- **When** le skill génère le SQL
- **Then** le résultat résout purchase_orders_idcor_supplier via id_mapping vers suppliers et agrège les montants

### AC2: Évolution achats par période
- **Given** la question "Évolution de mes achats par mois"
- **When** le skill génère le SQL
- **Then** le résultat utilise generate_series avec purchase_orders_order_date

### AC3: Achats par article
- **Given** la question "Quels articles j'achète le plus ?"
- **When** le skill génère le SQL
- **Then** le résultat agrège depuis purchase_orders_lines avec résolution article

### AC4: Factures fournisseurs
- **Given** la question "Montant total des factures fournisseurs ce trimestre"
- **When** le skill génère le SQL
- **Then** le résultat utilise purchase_invoices avec résolution idcor_supplier
