# User Story: Suivi réceptions et écarts

## Story

**As a** responsable logistique ou achats
**I want** suivre les réceptions et détecter les écarts avec les commandes
**So that** je puisse identifier les retards et les livraisons partielles fournisseurs

## Acceptance Criteria

### AC1: Réceptions par fournisseur
- **Given** la question "Combien de réceptions ce mois par fournisseur ?"
- **When** le skill génère le SQL
- **Then** le résultat utilise receipts avec résolution idcor_supplier

### AC2: Écarts commande/réception
- **Given** la question "Quelles commandes fournisseurs ont des écarts de réception ?"
- **When** le skill génère le SQL
- **Then** le résultat compare ordered_quantity vs received_quantity dans purchase_orders_lines

### AC3: Commandes en attente de réception
- **Given** la question "Quelles commandes fournisseurs attendent encore une réception ?"
- **When** le skill génère le SQL
- **Then** le résultat filtre sur receipt_status ou sur received_quantity < ordered_quantity
