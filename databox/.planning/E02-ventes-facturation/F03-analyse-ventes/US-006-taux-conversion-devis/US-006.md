# User Story: Taux de conversion devis→commandes

## Story

**As a** directeur commercial
**I want** connaître mon taux de conversion devis→commandes avec différentes ventilations
**So that** je puisse évaluer l'efficacité commerciale de mon équipe

## Acceptance Criteria

### AC1: Taux de conversion global
- **Given** la question "Quel est mon taux de conversion devis/commandes ?"
- **When** le skill génère le SQL
- **Then** le résultat calcule nb devis convertis (statut 2 ou 3) / nb total devis * 100

### AC2: Taux par commercial
- **Given** la question "Quel est le taux de conversion par commercial ?"
- **When** le skill génère le SQL
- **Then** le résultat ventile par commercial en résolvant quotes_idcor_sales_rep

### AC3: Taux par période
- **Given** la question "Comment évolue mon taux de conversion mois par mois ?"
- **When** le skill génère le SQL
- **Then** le résultat affiche le taux par mois avec generate_series

### AC4: Montant des devis convertis vs perdus
- **Given** la question "Quel est le montant des devis perdus ce trimestre ?"
- **When** le skill génère le SQL
- **Then** le résultat distingue les montants par statut (ouvert/commandé/partiellement)
