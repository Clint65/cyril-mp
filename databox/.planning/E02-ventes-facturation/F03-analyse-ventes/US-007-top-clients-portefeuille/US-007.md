# User Story: Top clients et analyse portefeuille

## Story

**As a** responsable commercial
**I want** identifier mes meilleurs clients et analyser mon portefeuille
**So that** je puisse concentrer mes efforts sur les comptes à plus fort potentiel

## Acceptance Criteria

### AC1: Top N clients par CA
- **Given** la question "Quels sont mes 10 meilleurs clients par CA cette année ?"
- **When** le skill génère le SQL
- **Then** le résultat affiche nom du client (via résolution idcor), CA, nombre de factures, trié par CA DESC

### AC2: Répartition du CA par client (Pareto)
- **Given** la question "Quelle part de mon CA font mes 20% meilleurs clients ?"
- **When** le skill génère le SQL
- **Then** le résultat calcule le cumul en % du CA total

### AC3: Clients sans commande récente
- **Given** la question "Quels clients n'ont pas commandé depuis 3 mois alors qu'ils commandaient régulièrement ?"
- **When** le skill génère le SQL
- **Then** le résultat identifie les clients ayant commandé dans les 12 mois précédents mais pas dans les 3 derniers mois

### AC4: Nouveaux clients
- **Given** la question "Combien de nouveaux clients ce trimestre ?"
- **When** le skill génère le SQL
- **Then** le résultat identifie les clients dont la première commande/facture est dans la période
