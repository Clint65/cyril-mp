# User Story: CA facturé avec gestion des avoirs

## Story

**As a** responsable financier
**I want** obtenir le CA facturé net (factures moins avoirs)
**So that** mes analyses de facturation reflètent la réalité comptable

## Acceptance Criteria

### AC1: CA facturé net avec invoices_sign
- **Given** la question "Quel est le CA facturé net ce mois ?"
- **When** le skill génère le SQL
- **Then** le SUM multiplie invoices_amount_lines_excl_vat par invoices_sign

### AC2: Détail factures vs avoirs
- **Given** la question "Combien d'avoirs ce trimestre et pour quel montant ?"
- **When** le skill génère le SQL
- **Then** le résultat sépare les factures (sign=+1) des avoirs (sign=-1) avec comptage et montants

### AC3: CA facturé par client avec avoirs déduits
- **Given** la question "CA facturé par client ce trimestre, avoirs déduits"
- **When** le skill génère le SQL
- **Then** le résultat par client utilise SUM(montant * invoices_sign) et résout idcor_account

### AC4: Évolution facturation mensuelle
- **Given** la question "Évolution de la facturation nette par mois"
- **When** le skill génère le SQL
- **Then** le résultat utilise generate_series et invoices_sign dans l'agrégation
