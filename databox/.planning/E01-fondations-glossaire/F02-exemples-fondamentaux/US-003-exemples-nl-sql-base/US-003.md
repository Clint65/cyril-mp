# User Story: Exemples NL→SQL de base

## Story

**As a** skill NL→SQL
**I want** disposer d'exemples few-shot couvrant les patterns fondamentaux Databox
**So that** je puisse m'appuyer sur des modèles validés pour générer du SQL correct

## Acceptance Criteria

### AC1: Exemple version courante
- **Given** la question "Liste tous les clients actifs"
- **When** l'exemple est écrit
- **Then** le SQL inclut INNER JOIN id_mapping avec end_date IS NULL

### AC2: Exemple résolution idcor
- **Given** la question "Quelles sont les factures du client X ?"
- **When** l'exemple est écrit
- **Then** le SQL résout invoices_idcor_account via id_mapping avec alias acc_im

### AC3: Exemple jointure article (cast TEXT)
- **Given** la question "Quel article se vend le plus en quantité ?"
- **When** l'exemple est écrit
- **Then** le SQL joint via id_product = articles_id_databox::text

### AC4: Exemple agrégation temporelle
- **Given** la question "Quel est mon CA par mois cette année ?"
- **When** l'exemple est écrit
- **Then** le SQL utilise generate_series pour inclure les mois sans données

### AC5: Exemple avec invoices_sign
- **Given** la question "Quel est le CA net facturé ?"
- **When** l'exemple est écrit
- **Then** le SQL multiplie les montants par invoices_sign dans le SUM

### AC6: Au moins 6 exemples validés par exécution
- **Given** les exemples écrits
- **When** ils sont exécutés sur la base
- **Then** chacun retourne des résultats cohérents sans erreur SQL
